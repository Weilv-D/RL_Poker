# RL Poker Playground

This directory contains interactive interfaces for playing against the RL AI.

## 1. Terminal UI (TUI)
Enhanced terminal interface with big cards.

```bash
# From project root
.venv/bin/python playground/play_tui.py --tui
```

## 2. Web Interface
Modern web-based GUI.

```bash
# From project root
.venv/bin/python playground/web_server.py
```
Open http://127.0.0.1:8000 in your browser.
